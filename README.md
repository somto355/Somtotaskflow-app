🚀 TaskFlow App

A clean and modern Task Management (Todo) App built with React + TypeScript.
Designed for productivity, simplicity, and a smooth user experience.

---

✨ Features

- ✅ Add and delete tasks
- ✅ Mark tasks as completed
- ✅ Persistent storage using Local Storage
- ✅ Clean and responsive UI
- ✅ Lightweight and fast

---

🛠️ Tech Stack

- ⚛️ React
- 🟦 TypeScript
- 🎨 CSS
- 💾 Local Storage API

⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/taskflow-app.git
cd taskflow-app

Install dependencies:

npm install

Run the app:

npm start
